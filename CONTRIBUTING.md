# Contributing

## Requirement

- [Rust](https://rustup.rs/) (stable + nightly)
- [pre-commit](https://pre-commit.com/)
- [cargo-udeps](https://github.com/est31/cargo-udeps)

## Getting started

```bash
git clone https://github.com/leroyguillaume/projectctl.git
cd projectctl
pre-commit install
```

## How to build

```bash
cargo build
```

## How to test

```bash
cargo test
```
